# SmartDirectory

This is the monorepo for `SmartDirectory`, a project developed for the HKUST x Sino Living Lab competition. The project is organized into the following directories:
 -   `/branding`: Automated branding agent that can develop a smart directory using artificial intelligence and machine learning.
 -   `/operations`: Automated workflow for modifying tenant structure for a building's e-directory.

### Contributing

To contribute to this project, check for relevant TODO tasks within each of the modules.
